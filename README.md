## Installation et démarrage

1. Cloner le repository
2. Installer les dépendances :
   ```bash
   npm install
   cd client && npm install
   ```
3. Démarrer le serveur :
   ```bash
   npm start
   ```
4. Dans un autre terminal, démarrer le client :
   ```bash
   cd client && npm start
   ```
